Variáveis

O que é uma variável?

uma variável é como uma caixinha onde voçe guarda um valor para usar depois.

variáveis basicas Javascript
  • Número;"let idade = "25;" "
  • Texto(String); "let nome = "Lucas" "
  • Booleano(Verdadeiro ou falso); "let estaChovendo = false;"
  • Array(Lista de valores); "let frutas = ["maça", "banana", "Laranja"];"
  • Objeto(conjunto de propriedades); let pessoa = { nome: "Ana", idade: 30 };
  • Constante(valor que não muda); "const PI = 3,14;"

como criar uma variável em javascript?

  • voçe pode usar tres palavras para criar variáveis:
    • palavras chaves
    • Let- pode mudar depois
    • const- não pode mudar
    • var- antiga, não é muito recomendada hoje
